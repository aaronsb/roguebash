# Multiclassing

## Prerequisites

To multiclass, you must meet the ability score prerequisites for both your current class and the new class.

| Class | Minimum Ability Score |
|-------|----------------------|
| Barbarian | Strength 13 |
| Bard | Charisma 13 |
| Cleric | Wisdom 13 |
| Druid | Wisdom 13 |
| Fighter | Strength 13 or Dexterity 13 |
| Monk | Dexterity 13 and Wisdom 13 |
| Paladin | Strength 13 and Charisma 13 |
| Ranger | Dexterity 13 and Wisdom 13 |
| Rogue | Dexterity 13 |
| Sorcerer | Charisma 13 |
| Warlock | Charisma 13 |
| Wizard | Intelligence 13 |

## Proficiencies Gained

When you multiclass into a new class, you gain only some of that class's proficiencies:

| Class | Proficiencies Gained |
|-------|---------------------|
| Barbarian | Shields, simple weapons, martial weapons |
| Bard | Light armor, one skill of your choice, one musical instrument |
| Cleric | Light armor, medium armor, shields |
| Druid | Light armor, medium armor, shields (no metal) |
| Fighter | Light armor, medium armor, shields, simple weapons, martial weapons |
| Monk | Simple weapons, shortswords |
| Paladin | Light armor, medium armor, shields, simple weapons, martial weapons |
| Ranger | Light armor, medium armor, shields, simple weapons, martial weapons, one skill from the class list |
| Rogue | Light armor, one skill from the class list, thieves' tools |
| Sorcerer | — |
| Warlock | Light armor, simple weapons |
| Wizard | — |

## Multiclass Spellcasting

If you have spellcasting from multiple classes, determine your spell slots using the Multiclass Spellcaster table.

### Determining Caster Level
Add together:
- **Full caster levels** (Bard, Cleric, Druid, Sorcerer, Wizard): add full class level
- **Half caster levels** (Paladin, Ranger): add half class level (rounded down)
- **Third caster levels** (Eldritch Knight Fighter, Arcane Trickster Rogue): add one-third class level (rounded down)

Use the combined total with the full caster spell slot table.

## Pact Magic

Warlock Pact Magic slots are tracked separately from other spellcasting, but you can use Pact Magic slots to cast spells from other classes and vice versa.

## Spells Known and Prepared

Each class's spells are determined using your level in **that specific class**, not your total character level. A 5th-level Cleric / 3rd-level Wizard prepares cleric spells as a 5th-level cleric (including 3rd-level cleric spells) and wizard spells as a 3rd-level wizard (including 2nd-level wizard spells), even though their multiclass spell slots go up to 4th level.

## Key Multiclassing Interactions

- **Extra Attack** does NOT stack. Multiple sources of Extra Attack don't give you additional attacks.
- **Unarmored Defense** can only be gained from one class.
- **Channel Divinity** from multiple sources doesn't grant extra uses — you get the options but share the use count.
- **Cantrips** scale with total character level, not class level.
- **Class features** that scale with "class level" use only the levels in that specific class.
- **HP and Hit Dice** are calculated per class (roll each class's hit die when gaining a level in that class).
