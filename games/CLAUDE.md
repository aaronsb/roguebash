# games/ — Campaign Data

Each campaign lives in its own subdirectory with full state tracking: player characters, world building, session logs, quests, combat, economy, relationships, and narrative arcs. Use `/session new {name}` to create a new campaign.

See the root `CLAUDE.md` for the full campaign directory structure template.
